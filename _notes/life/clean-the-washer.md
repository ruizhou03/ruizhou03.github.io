# 1. 问题

洗衣机是用来洗衣服的——它自己也需要洗吗？打开机门有时闻到一股酸臭味或霉味，是怎么回事？洗衣液都倒进去那么多，里面应该越洗越干净才对啊？为什么衣服洗完会有黑色絮状物？洗衣机本身怎么清洁、多久一次、用什么？

# 2. 结论先行

**洗衣机自己也需要定期清洁，而且比想象中重要得多**——

研究测试发现，**普通家用洗衣机内部潮湿表面的菌落数有时可超过马桶座圈**——主要是因为：

1. **常年潮湿** + **30-40 ℃** 温度（适合微生物生长）
2. **洗衣液 + 皮脂 + 死皮屑** 在橡胶圈、洗涤剂盒、内胆夹层堆积形成 **生物膜（biofilm）**
3. **滚筒洗衣机**比波轮更易长霉（橡胶门封是重灾区）

**频率**：

- **每月一次**深度清洁（运行清洁程序 / 加专用清洁剂）
- **每次用完**留门通风 30 分钟
- **每周**擦拭洗涤剂盒、橡胶门封内圈
- **每 6-12 个月**深度拆洗（洗涤剂盒、过滤器、橡胶圈）

**用什么洗**：

- **洗衣机槽专用清洁剂**（最方便、最对路）
- **白醋 + 小苏打**（家用替代，但效果略弱）
- **含氧漂白剂**（OxiClean / Vanish，去霉斑很好）
- **绝对不用**含氯漂白剂（84 / Clorox）—— 长期会腐蚀洗衣机金属和橡胶

# 3. 科学原理

## 3.1 洗衣机为什么会脏

洗衣机是个矛盾的存在——它是清洁工具，但**自己最容易脏**：

**1. 温暖 + 潮湿 = 微生物天堂**

洗衣机内部一年到头处于：

- 平均温度 25-35 ℃（夏天更高）
- 湿度 80-100%
- 黑暗、不通风

正是细菌、霉菌、酵母最爱的环境。

**2. 残留有机物喂养微生物**

每次洗衣，**留下一堆“食物”给微生物**：

- 衣物上的皮脂、汗液、死皮屑
- 洗涤剂残留（特别是柔顺剂——油性，最易残留）
- 织物纤维碎屑（毛巾、衣物脱落的小毛）
- 食物残渣（衣服上沾的菜汁、果汁）

**3. 死角多**

洗衣机不是光滑的桶，有大量隐藏空间：

- **橡胶门封**（滚筒）—— 内圈的褶皱里堆积水 + 残渣
- **洗涤剂盒** —— 洗涤剂结块 + 反向流水形成黏液
- **滚筒夹层 / 内胆外壁** —— 你看不到的内层堆积污垢
- **排水过滤器**（滚筒底部小盖板）—— 拦截毛发、纸巾、硬币
- **内胆顶端** —— 翻动时不接触水，长年发霉

时间一长，污垢和微生物形成 **生物膜**——一层粘稠的”细菌泥”，黏附力极强：

- 普通水冲不掉
- 洗衣液也很难穿透
- 越长越厚 → 隐藏更多微生物 → 异味越重 → 衣服越洗越臭

## 3.2 常见的“洗衣机征兆”

如果你发现下面任何一条，洗衣机已经该深度清洁了：

| 现象 | 原因 |
|---|---|
| **打开机门有酸臭 / 霉味** | 橡胶门封 / 内胆长霉 |
| **刚洗完的衣服有怪味** | 生物膜污染干净衣物 |
| **衣服上出现黑色絮状物** | 内胆夹层霉斑脱落 |
| **白色衣物洗完有灰渍** | 洗涤剂结块 + 微塑料 + 灰尘 |
| **洗涤剂盒发黏 / 长霉** | 洗涤剂残留 + 反流积水 |
| **排水变慢 / 异响** | 排水过滤器堵塞（毛发、硬币、纸巾）|
| **机门橡胶圈发黑发绿** | 霉菌生长 |
| **衣物有“湿洗澡味”** | 微生物代谢产物（吲哚等） |

## 3.3 滚筒 vs 波轮：哪个更脏

**滚筒**（前置式）：

- 优点：耗水少
- 缺点：**橡胶门封内圈 + 内胆夹层**是污垢重灾区
- **更易长霉**——因为水分残留多

**波轮**（顶置式）：

- 优点：水流冲刷有自洁作用
- 缺点：**内桶外壁 / 桶底**是污垢主战场
- **平均霉菌量略低**于滚筒，但仍需定期清洁

**结论**：**滚筒比波轮更需要勤清洁**——特别是门封那圈橡胶要每周擦。

# 4. 实践建议

## 4.1 日常护理（每天 / 每周做的小事）

**每次用完**：

1. **打开机门**（让里面通风干燥）—— 这是最重要的一条习惯
2. **取出洗涤剂盒抽屉**，擦干（如果可拆）
3. **擦干橡胶门封内圈**（用纸巾或毛巾）
4. **不要把湿衣服留在机里过夜**——立刻晾晒或烘干

**每周一次**：

1. 用湿布 + 中性洗涤剂擦**洗涤剂盒**（特别是柔顺剂格）
2. 用湿布 + 白醋擦**橡胶门封内圈**（注意褶皱里的黏液）
3. 检查**排水过滤器**（一般在滚筒前下方的小盖板）

## 4.2 月度深度清洁

每月一次（北方硬水地区可以更勤），用以下方法之一：

**方法 1：洗衣机专用清洁剂（最推荐）**

市面常见品牌：威王、白猫、净安、汰渍专用、Affresh、Tide Washing Machine Cleaner（美国）。

主要成分：**过碳酸钠**（释放过氧化氢）+ **柠檬酸 / 螯合剂** + **酶**。

操作：

1. 空机（不放衣物）
2. 倒入清洁剂（按包装指示，一般 1 包 / 1 小瓶）
3. 选择**「筒清洁」/「自清洁」/「桶自洁」**程序（如果有）；没有的话用最长的**「热水 / 高温」长程序 + 大水量**（欧标滚筒能到 60-90 ℃；美标顶置波轮 hot 档约 54 ℃ / 130 °F，能跑多热就跑多热）
4. 程序结束后，再用清水**额外漂洗一次**（避免残留）
5. 擦干内胆 + 门封 + 留门通风

**方法 2：白醋（家用替代）**

用法：

1. 倒入 **2 杯（500 ml）白醋** 到主洗格
2. 选择 **60 ℃ 高温长程序**
3. 程序结束后再加 **半杯小苏打** 在内胆里
4. 再次跑 60 ℃ 程序
5. 漂洗一次

注意：**白醋 + 小苏打不能同时加**——会立即反应放出 $\text{CO}_2$ + 失去清洁力。**先白醋一遍，再小苏打一遍**。

效果：比专用清洁剂略弱，但对**轻度污垢 + 异味**已经够。优点是几乎免费 + 食品级安全。

**方法 3：含氧漂白剂**

OxiClean / Vanish 加 1 杯（200 g）+ 60 ℃ 长程序。

适合**已经有明显霉斑**的情况——氧气漂白能直接氧化霉斑色素。

**绝对不要用**：

- ❌ **含氯漂白剂（84 / Clorox）**：会腐蚀不锈钢内胆 + 橡胶门封——**长期使用洗衣机会缩短寿命**
- ❌ **盐酸 / 强酸清洁剂**：腐蚀金属
- ❌ **柔顺剂当清洁剂用**：柔顺剂自己就是污垢源头

## 4.3 季度 / 半年深度拆洗

**洗涤剂盒**（每季度）：

1. 拆下洗涤剂盒（大多数洗衣机能整体抽出）
2. 在水池里用旧牙刷刷
3. 浸泡 30 分钟在白醋水里
4. 冲洗、晾干、装回

**排水过滤器**（每半年）：

> ⚠️ 操作前先排水：某些滚筒过滤器底部有一个小排水管，先把那里的水排掉再开过滤器，避免脏水喷出

1. 找到滚筒前下方的小盖板（机型不同位置略异）
2. 拧开过滤器（注意：可能会有积水，先放盆接住）
3. 取出**毛发、纸巾、硬币、纽扣**等异物
4. 用刷子刷干净滤网
5. 装回拧紧

**橡胶门封内圈**（每月深度擦一次）：

1. 用一只手把橡胶圈往外翻开
2. 用旧牙刷 + 含氧漂白剂或白醋擦褶皱
3. 用湿毛巾擦干净
4. 用纸巾吸干
5. 留门通风

## 4.4 怎么避免洗衣机变脏

**习惯改变**（治本）：

- ✅ **每次用完留门通风** 30 分钟以上
- ✅ **湿衣服立刻拿出来**——不要留在机里过夜
- ✅ **洗涤剂用量按推荐**——多倒了反而残留更多
- ✅ **少用柔顺剂**或**完全不用**——柔顺剂是洗衣机最大的污垢来源
- ✅ **避免长期冷水洗**——偶尔（每月）用 60 ℃ 跑一次清洁

**频次最佳实践**：

- 每天：用完留门通风
- 每周：擦门封 + 洗涤剂盒
- 每月：跑一次清洁程序
- 每季度：洗洗涤剂盒
- 每半年：清排水过滤器
- 每年：考虑专业上门清洁（如果是老机器）

## 4.5 专业清洁（什么时候考虑）

**家用专业清洁**（淘宝 / 京东 / 美团）：

- 价格：¥80-300（看城市 / 上门）
- 服务内容：拆机、深度清洁内胆夹层（你看不到的那一层）、清洗排水管、消毒
- 频率：**老机器（$\geq 3$ 年）每年一次**；新机器不用

**什么时候必须叫专业**：

- 洗衣机用了 3 年以上从来没清过
- 衣服反复出现黑色絮状物
- 异味怎么洗都去不掉
- 拆开看见**内胆夹层有黑色霉斑**

用 5-8 年没拆洗过的老机器，第一次专业上门拆开内胆夹层时，**取出来的污垢量往往比人们预想的要多**。这并不说明日常洗衣“没洗干净”——夹层是水流冲不到的死区，靠跑程序解决不了，需要物理拆机才能清。

## 4.6 几个误区

**误区 1：「洗衣机一直在用洗衣液冲，应该自己很干净」**

错。洗涤剂浓度低 + 温度低 + 残留多——内胆夹层等“死区”几乎得不到清洁，反而成为污垢温床。

**误区 2：「闻不到异味就是干净」**

错。生物膜在异味出现前就已经形成；异味是已经严重污染的信号。

**误区 3：「84 / Clorox 灌进去最干净」**

错。含氯漂白剂会腐蚀不锈钢和橡胶——短期看上去干净，长期会缩短洗衣机寿命 + 把橡胶圈搞得发黑发硬。

**误区 4：「洗衣机自洁程序就够了」**

部分对。**自洁程序 + 加清洁剂**才完整。光跑空程序作用很有限——没有化学攻击，污垢黏附力太强。

**误区 5：「滚筒洗衣机不用清洁因为它很现代」**

恰恰相反。**滚筒比波轮更需要清洁**——橡胶门封是霉菌温床。买滚筒就要做好“每月清一次”的心理准备。

**误区 6：「关上洗衣机门保持密封」**

完全错。**关门 = 留住湿气 = 长霉**。**用完留门通风**是洗衣机长寿和卫生的关键习惯之一。
