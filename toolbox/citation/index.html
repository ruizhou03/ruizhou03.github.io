🔖 引用格式转换器

BibTeX → APA / Chicago / MLA / Harvard / Vancouver

输出
在左边粘贴 BibTeX，这边自动渲染
支持的字段 / 已知限制

支持的 BibTeX 类型：@article、@book、@inproceedings、@incollection、@inbook、@phdthesis、@mastersthesis、@techreport、@misc。其他类型按 misc 处理。

识别的字段：author、title、journal / booktitle、year、volume、number / issue、pages、publisher、address、editor、doi、url、edition、chapter、institution、school、note。

多作者格式：用 and 分隔，如 author = {Smith, John and Doe, Jane}。也支持 {First Last} 形式（自动识别 last name）。

已知限制：

  • 不解析 LaTeX 转义字符（\&, {\"o} 等保持原样）—— 想要完美渲染请用 bibtex-tidy 等专业工具
  • 不支持 BibTeX @string 替换
  • 不识别 crossref 字段（会忽略）
  • format 输出按各 style guide 主流写法，但实际期刊可能有细节差异

如需引用进 academic paper，强烈建议用 Zotero / Mendeley 这类工具配合期刊模板生成；本工具适合快速估个样子或轻量场景（博客、PPT、邮件）。
