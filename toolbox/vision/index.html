👁️ 视力自测

Vision Self-Check · 浏览器内的家庭筛查

三步开测：
  1. 设置浏览器缩放为 100%（自动检测）
  2. 校准屏幕物理尺寸（多数 iPhone/iPad 自动；其它用银行卡）
  3. 选好观看距离 → 蒙一只眼开测

数据全本地，不联网，不上传。可选保存历史到本机。

⚠️ 这是筛查/趣味工具，不替代眼科检查。 任何疑似异常（视物变形、闪光、暗点、急性视力下降等）请立即就医。

1. 浏览器缩放

⏳检测中…

按 Cmd+0（Mac）或 Ctrl+0（Windows / Linux）重置缩放，本步骤会自动通过。

检测精度有限。如果你确定缩放是 100%，可以跳过这一步。手机系统的字体缩放不影响测试（我们用 px 而非 rem）。

2. 屏幕校准

✅ 检测到设备：
屏幕物理像素密度已识别（PPI ≈ ），无需校准。
⚠️ 没在数据库里匹配到你的设备。请改用「信用卡」或「手动输入」。

把任意银行卡 / 驾照 / 学生证贴到屏幕上，拖滑块让矩形和卡片完全重合。

屏幕像素密度：3.74 px/mm（卡宽对应 320 px）

如果你知道屏幕的对角线英寸（笔记本贴纸 / 电视参数），直接填。

4. 测前准备

  • ✓ 室内光线充足，不要逆光
  • ✓ 准备一张纸 / 不透光的杯子，盖住一只眼。不要紧闭，不要按压眼球，否则数据会受影响
  • ✓ 流程：先测右眼（盖左眼），再测左眼（盖右眼）
  • ✓ 下面选裸眼 / 戴镜（影响结果解读）
