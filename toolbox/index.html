🧰 生活百宝箱

Toolbox

🎲
遇事不决
权重转盘抽签器。几个人吃饭选不出哪家、看什么电影时用，可以给选项不同权重，可保存方案。
已上线
👥
随机分组器
N 个人随机分成 K 组（或每组 M 人），适合 TA 分 section、聚餐 AA、桌游开局。
已上线
🎤
随机点名
名单导入后随机抽人，支持「不重复模式」自动遍历完所有人。上课提问 / 组会汇报顺序。
已上线
💱
汇率 + 单位换算
USD ↔ RMB 等主流货币，以及 lb/kg、℉/℃、in/cm。三层 fallback：实时 → 缓存 → 兜底，离线也能用。
已上线
📈
复利 / 现值
本金 + 月供 + 利率 → 终值曲线；或未来金额 → 现值贴现。401k 模拟、HYSA 攒钱目标可视化。
已上线
📊
报税档位估算（US / CN）
🇺🇸 美国 federal+州税 / 🇨🇳 中国综合所得+五险一金+专项附加扣除。看分段税率、边际/实际税率、税后到手。顶部一键切国家。
已上线
⏳
倒计时面板
添加自定义事件（考试、deadline、生日、回国机票），多张卡片并列显示「还剩 X 天 Y 小时」或「已过去 X 天」。剩 ≤ 3 天高亮。
已上线
📅
X 年前的今天
翻翻历年同一天写过的文章。带写作热力图——一年 365 天哪天写了几篇，一目了然。
已上线
🌳
种树
类似 Forest / 番茄钟。点开始 → 一棵树跟着 25 分钟一起长大。离开 tab 超过 15 秒树会枯萎。每棵树都进你的森林。
已上线
🔖
引用格式转换器
粘贴 BibTeX → 输出 APA / Chicago / MLA / Harvard / Vancouver 6 种格式。多 entries 批量处理，一键复制。
已上线
🎯
目标进度跟踪
长期目标 + 打卡。瘦 5 kg / 读 12 本书 / GRE 写作 320 分。折线图对比实际曲线 vs 理想线，看是 on track / 落后 / 超前。
已上线
📖
生词本（间隔重复）
单词 + 释义 + 例句 + 标签。Leitner 5 档 box 间隔重复，答对升档答错回 Box 1。3D 翻牌动画，键盘快捷键。
已上线
