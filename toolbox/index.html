🧰 生活百宝箱

Toolbox

💱
汇率 + 单位换算
USD ↔ RMB 等主流货币，以及 lb/kg、℉/℃、in/cm。三层 fallback：实时 → 缓存 → 兜底，离线也能用。
已上线
⏳
倒计时面板
添加自定义事件（考试、deadline、生日、回国机票），多张卡片并列显示「还剩 X 天 Y 小时」或「已过去 X 天」。剩 ≤ 3 天高亮。
已上线
🎲
遇事不决
权重转盘抽签器。几个人吃饭选不出哪家、看什么电影时用，可以给选项不同权重，可保存方案。
已上线
👥
随机分组器
N 个人随机分成 K 组（或每组 M 人），适合 TA 分 section、聚餐 AA、桌游开局。
已上线
🌳
种树
类似 Forest / 番茄钟。点开始 → 一棵树跟着 25 分钟一起长大。离开 tab 超过 15 秒树会枯萎。每棵树都进你的森林。
已上线
🎯
目标进度跟踪
长期目标 + 打卡。瘦 5 kg / 读 12 本书 / GRE 写作 320 分。折线图对比实际曲线 vs 理想线，看是 on track / 落后 / 超前。
已上线
📖
生词本（间隔重复）
单词 + 释义 + 例句 + 标签。Leitner 5 档 box 间隔重复，答对升档答错回 Box 1。3D 翻牌动画，键盘快捷键。
已上线
📈
复利 / 现值
本金 + 月供 + 利率 → 终值曲线；或未来金额 → 现值贴现。401k 模拟、HYSA 攒钱目标可视化。
已上线
📊
报税档位估算（US / CN）
🇺🇸 美国 federal+州税 / 🇨🇳 中国综合所得+五险一金+专项附加扣除。看分段税率、边际/实际税率、税后到手。顶部一键切国家。
已上线
🎤
随机点名
名单导入后随机抽人，支持「不重复模式」自动遍历完所有人。上课提问 / 组会汇报顺序。
已上线
🔖
引用格式转换器
粘贴 BibTeX → 输出 APA / Chicago / MLA / Harvard / Vancouver 6 种格式。多 entries 批量处理，一键复制。
已上线
🍉
合成大西瓜
物理小游戏。同种水果碰撞合并升级，从樱桃一路合到西瓜。基于 Matter.js，支持鼠标/触屏/键盘。
已上线
👁️
视力自测
浏览器内的家庭视力筛查：信用卡校准 + 距离预设 + Tumbling E 自适应 + 散光盘。左右眼分测，结果以 20/X 给。仅作筛查参考，不替代验光。
已上线
🥁
节拍器
练琴 / 节奏训练 / 跑步配速。BPM 30–300、常见拍号一键切换、第一拍重音、Tap Tempo 反推 BPM。Web Audio look-ahead 调度，长时间不漂移。
已上线
🎵
音高测量
麦克风实时检测或上传音频文件 → autocorrelation 算法识别音高，给音名 + 频率 + cents 偏差，文件模式画 pitch contour。声乐 / 调音 / 跟唱跑调自查。麦克风音频只在浏览器内处理，不上传。
已上线

🎮 小游戏

纯离线，打发碎片时间
🎯
2048
滑动合并相同数字凑 2048。3×3 / 4×4 / 5×5 三种棋盘，撤销一步、自动存档、按尺寸分别记最高分，通关后还能继续到 4096。
已上线
🐍
贪吃蛇
经典街机款。方向键 / WASD / 移动端虚拟方向键，慢中快三档速度各自记最高分。空格暂停。
已上线
💣
扫雷
Windows 经典三难度（9×9 / 16×16 / 16×30）。首次点击保证不踩雷，空格自动 flood-fill。桌面右键插旗，移动端长按 0.4s 插旗，按难度记最佳用时。
已上线
⚫
五子棋
15×15 木纹棋盘。双人同机或人机对战（可选执黑/白），启发式 AI 评估活四 / 冲四 / 活三等棋型。支持悔棋、胜负计数。
已上线
🎓
Deadline 大逃亡
Chrome 断网恐龙的本地化版本。博士生跑酷，跳过文献综述纸堆，钻过 Reviewer 2 的飞邮件。变高跳 + 蹲、速度逐渐加快、撞到就 GG，最高分本机存。
已上线
