打砖块

挥动挡板，把弹球弹回去，砸光所有砖块
--
历史最高分
用鼠标 / 手指拖动，或 ← → 移动挡板 · 空格 发球 / P 暂停
游戏规则

移动挡板把弹球弹回去，砸掉所有砖块即可进入下一关。挡板边缘击球反弹角更斜，可借此控制球路。砖块越靠上分越高，关数越高砖墙越密、球速越快。掉球一次扣一条命，三命用完游戏结束。

操作：鼠标 / 触屏拖动挡板，或键盘 ← →。点击画面或按 空格 发球，P / 暂停按钮 暂停。
