\section*{如何阅读本书}

这本书写给正在念经济学博士、却时常被数学``卡一下''的人——包括我自己。经济学的核心课程（微观、宏观、计量）会不断借用数学工具：计量里大量的线性代数，微观里的最优化与比较静态，宏观里的动态规划，决策论与博弈论里的概率、测度与贝叶斯更新。这些工具往往散落在不同的数学课本里，各讲各的、各有各的记号，且大多是为数学家而非经济学家写的——它们把``严格''放在第一位，却很少告诉你\textbf{这个工具是用来干什么的}。本书想做的，正是把这些工具集中到一处，并且\emph{始终站在使用者的角度}来讲。

\subsection*{每个知识点都按``三件套''来写}

本书不追求数学课本那种环环相扣的逻辑链条；相反，它以\textbf{工具}为单位来组织，每一个工具都尽量回答三个问题：

\begin{itemize}
    \item \textbf{它解决什么问题（用处）}——先说清楚经济学家为什么需要它，它在你将来会遇到的哪类问题里登场。
    \item \textbf{它从哪里来（推导）}——给出关键的推导与直觉，让你看清这个结论不是从天而降，而是可以自己重建的。证明以``讲明白''为标准，少数极重的纯数学构造只陈述并给直觉、不逐行搭建。
    \item \textbf{它以后用在哪（去处）}——指出它在微观、宏观、计量乃至后续章节里的具体落点，把孤立的工具接回经济学的主干。
\end{itemize}

正因如此，各章之间\textbf{不强求严丝合缝的先后衔接}：你可以按需跳读，遇到不熟的前置概念时再回头翻查。每章开头会点明它依赖哪些前面的工具。

\subsection*{彩框颜色约定}

\begin{itemize}
    \item \tcbox[colback=green!10!white, colframe=green!10!white, size=fbox]{\textbf{绿色：定义}}——我们赖以构建的对象与概念。
    \item \tcbox[colback=cyan!10!white, colframe=cyan!10!white, size=fbox]{\textbf{蓝色：定理 / 命题}}——核心结论，通常附证明。
    \item \tcbox[colback=pink!20!white, colframe=pink!20!white, size=fbox]{\textbf{粉色：假设}}——一个论证所维持的前提。
    \item \tcbox[colback=violet!8!white, colframe=violet!8!white, size=fbox]{\textbf{淡紫：要点框}}——对工具性的提炼，或一句话点透的关键直觉。
    \item 全书还穿插着\textbf{例}（worked examples）与\textbf{注}（remarks）。
\end{itemize}

\subsection*{记号约定}

向量用粗体小写，如 $\vc{x},\vc{\beta}$；矩阵用粗体大写，如 $\mat{A},\mat{H}$；转置记为 $\mat{A}\T$，逆记为 $\inv{\mat{A}}$。实数集记 $\RR$，$n$ 维欧氏空间记 $\RR^n$。梯度记 $\grad f$，Hessian 记 $\grad^2 f$；映射 $\mat F$ 关于变量组 $\vc x$ 的 Jacobian 记 $\D_{\vc x}\mat F$。概率测度记 $\PP$，期望记 $\E{\cdot}$，方差、协方差记 $\Var{\cdot}$、$\Cov{\cdot}{\cdot}$；$\sigma$-代数等用花体 $\cF,\cB$。依概率、依分布收敛分别记 $\pto$、$\dto$。各章若另有局部记号，会在该章开头声明。
